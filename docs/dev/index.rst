
For Contributors
================


This guide is intended for developers who want to contribute to the Truelearn project.

In this guide, you will find information about how to install the project from source, how to run the tests, and how to build the documentation.

Ignore this for now ______________________________________________________________
in regards to venv, you can use the following command to create a virtual environment:
(Note some IDEs have their own virtual environments, so you may not need to do this)
python3 -m venv venv
then activate it with:
source venv/bin/activate
then install the requirements with:
pip install truelearn
